# 大家的日本語 初級1 — 單字與句型視覺化整理

## 概述

製作一個單一 HTML 檔案，整理「大家的日本語 初級1」第 1～6 課的單字與句型（文型），以教科書風格呈現，具備互動式學習功能。

## 目標使用者

日文初學者，目前學到第 6 課，需要一份可視化的複習工具。

## 架構

### 單頁面 + 側邊欄導航

- 所有 6 課內容在同一個 HTML 檔案中
- 無外部依賴（CSS/JS 全部內嵌），可離線使用
- 直接用瀏覽器開啟即可

## 視覺風格

### 教科書風格

- **色調**：暖色背景 `#fdf8f0`，側邊欄 `#f5ede0`
- **重點色**：紅色 `#c0392b` 用於課程標題左邊線、句型公式、section 標題底線
- **分隔線**：點線 (dotted) 分隔單字列
- **字型**：系統字型，日文優先（Hiragino Sans、Noto Sans JP 等）

### 詞性標籤

以小圓角標籤顯示詞性，不同類別用不同底色：

| 詞性 | 底色 | 文字色 |
|------|------|--------|
| 名詞 | `#fef3e8` | `#8b5e3c` |
| 動詞I/II/III | `#e8f0fe` | `#3a6bb5` |
| い形容詞 | `#e8f4f0` | `#2e7d5e` |
| な形容詞 | `#f0e8f4` | `#6b3a9e` |
| 副詞 | `#fef8e8` | `#8b7a3c` |
| 助詞/其他 | `#f0f0f0` | `#666666` |

## 內容結構

### 單字區（単語）

每個單字一列，包含：

1. **日文**：平假名/片假名表記（大字）
2. **漢字**：若有漢字寫法，以括號標注在日文旁
3. **詞性標籤**：小圓角 badge
4. **中文翻譯**：預設可見，可全域切換隱藏（用於自我測驗）
5. **發音按鈕**：🔊 圖示，點擊用 Web Speech API 朗讀

### 句型區（文型）

每個句型以卡片呈現，包含：

1. **句型公式**：如「N1 は N2 です」，紅色粗體
2. **中文說明**：句型的中文解釋
3. **例句**（2-3 句）：
   - 日文例句（附發音按鈕 🔊）
   - 中文翻譯（可隱藏）
4. **變化表**：

| 形式 | 句型 |
|------|------|
| 肯定 | N1 は N2 **です** |
| 否定 | N1 は N2 **じゃ ありません** |
| 疑問 | N1 は N2 です**か** |

變化中的關鍵部分用顏色標記（肯定=紅、否定=綠、疑問=藍）。

## 互動功能

### 1. 點擊顯示/隱藏翻譯

- 側邊欄有「顯示全部翻譯」/「隱藏全部翻譯」按鈕
- 也可點擊單一翻譯文字切換該項的顯示/隱藏
- 隱藏時翻譯區域顯示為模糊或底線佔位

### 2. 搜尋/篩選

- 側邊欄搜尋框，即時篩選（keyup 事件）
- 可搜尋日文、中文翻譯
- 搜尋時高亮匹配文字
- 搜尋結果跨課顯示，未匹配的課程自動收合（隱藏整個課程區塊）

### 3. 語音朗讀

- 使用 Web Speech API (`speechSynthesis`)
- 語言設為 `ja-JP`
- 點擊 🔊 按鈕朗讀對應的日文文字
- 語速適中（rate: 0.8），適合初學者

### 4. 課程導航

- 左側固定側邊欄，點擊課程名稱平滑捲動到對應區域
- 捲動時自動高亮當前所在課程（Intersection Observer）

## 技術實作

- **純 HTML/CSS/JS**：無框架依賴
- **單一檔案**：所有 CSS 和 JS 內嵌在 HTML 中
- **響應式設計**：手機上側邊欄轉為頂部下拉選單（斷點 < 768px）
- **Web Speech API**：瀏覽器原生語音合成，無需外部服務。若瀏覽器不支援，隱藏 🔊 按鈕

## 課程內容範圍

### 第1課 わたしは マイク・ミラーです
- 單字：自我介紹相關（わたし、あなた、せんせい、がくせい、かいしゃいん 等）
- 句型：N は N です / N は N じゃ ありません / N は N ですか

### 第2課 これは 本です
- 單字：指示代名詞、日常物品（これ、それ、あれ、本、辞書、かばん 等）
- 句型：これ/それ/あれ は N です / この/その/あの N

### 第3課 ここは 食堂です
- 單字：場所相關（ここ、そこ、あそこ、食堂、トイレ、階段 等）
- 句型：ここ/そこ/あそこ は N です / N は どこですか

### 第4課 今 何時ですか
- 單字：時間、日期相關（今、何時、何分、午前、午後 等）
- 句型：今 N時 N分です / N から N まで / N曜日

### 第5課 甲賀へ 行きます
- 單字：交通、移動相關（行きます、来ます、帰ります、バス、電車 等）
- 句型：N へ 行きます/来ます/帰ります / N で 行きます

### 第6課 いっしょに 京都へ 行きませんか
- 單字：動作動詞（食べます、飲みます、見ます、買います 等）
- 句型：N を V ます / N を V ませんか / V ましょう
