# みんなの日本語 — 単語と文型

互動式日語學習網站，整理《みんなの日本語》初級 I・II（第1課～第50課）的單字與文型。

## 功能

- 單字列表（含假名、漢字、詞性標籤、繁體中文翻譯）
- 文型/文法卡片（含例句與中文翻譯）
- 點擊翻譯隱藏/顯示（模糊效果，方便自我測驗）
- 日語語音朗讀（Web Speech API）
- 即時搜尋/篩選（支援假名、漢字、中文）
- 側邊欄快速跳轉
- 響應式設計（手機/桌面皆可使用）

## 結構

```
├── index.html      # 首頁（導覽頁）
├── shokyuu1/
│   └── index.html  # 初級 I（第1課～第25課）
└── shokyuu2/
    └── index.html  # 初級 II（第26課～第50課）
```

## 部署

靜態網站，透過 Cloudflare Pages 部署。推送至 `main` 分支即自動部署。
